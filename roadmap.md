# Roadmap

## Upcoming Features

### Pivot Table
- Enable advanced data summarization and analysis using pivot tables.

### Integrated Charts
- Visualize grid data with built-in charting capabilities (bar, line, pie, etc.).

### AI Features

#### Predictive Filtering
- Use AI to suggest relevant filters based on user behavior and data trends.

#### Smart Sorting
- Automatically sort data using intelligent algorithms that learn from user actions and data patterns.

#### Anomaly Detection
- Detect and highlight unusual or outlier data points using machine learning techniques.
